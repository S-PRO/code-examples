### Timer component 

Shows hours, minutes and seconds in real time. It will start when app starts and also update counted time when app in background or inactive.

### Techonologies used:

- [Last Features of ECMAScript 6/7+ standards](https://www.ecma-international.org/ecma-262/7.0/index.html)
- [MobX](https://mobx.js.org/)
- [Flow](https://flow.org/)
- [Moment](https://momentjs.com/)
- [Animations](https://github.com/oblador/react-native-animatable)
